# 🎟️ CRUD Ticket Booking System

A **Ticket Booking System** built with CRUD (Create, Read, Update, Delete) functionality for managing events and ticket reservations.

## ✨ Features

- **📝 Create Events** – Add new events with details like name, date, venue, and price.  
- **👀 View Events** – Browse all available events in a user-friendly list.  
- **✏️ Update Events** – Modify event details or correct mistakes.  
- **🗑️ Delete Events** – Remove outdated or canceled events.  
- **🎫 Book Tickets** – Reserve tickets for selected events.  
- **📊 Admin Dashboard** – Manage events and bookings efficiently.

## 🚀 Getting Started

### Prerequisites
- Node.js & npm installed  
- MongoDB Atlas account or local setup  

### Installation
1. **Clone the repo**  
   ```bash
   git clone https://github.com/yourusername/ticket-booking-system.git
   ```
2. **Install dependencies**  
   ```bash
   npm install
   ```
3. **Set up environment variables**  
   Create a `.env` file and add:
   ```
   MONGODB_URI=your_mongodb_connection_string
   PORT=3000
   ```
4. **Run the app**  
   ```bash
   npm start
   ```
5. **Open in browser**  
   Visit `http://localhost:3000`
